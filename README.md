# Hospital Emergency Room Analysis | Power BI

## About the Project

Developed an interactive Power BI dashboard to analyze Emergency Room operations, monitor patient flow, evaluate service performance, and provide actionable insights for improving hospital efficiency.

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## Project Highlights

- Built an interactive dashboard using **9,216 patient records**, providing a centralized view of Emergency Room performance.
- Developed KPI measures to monitor **9,216 Total Patients**, **35.3 Minutes Average Wait Time**, **4.99 Average Patient Satisfaction Score**, and **3,816 Department Referrals**.
- Created **four interactive report pages** including **Monthly View, Consolidated View, Patient Details, and Key Takeaways** to support operational and management reporting.
- Designed dynamic dashboards using **DAX, KPI Cards, Slicers, and Interactive Visualizations** to enable detailed analysis across multiple patient and operational metrics.
- Analyzed patient admissions, demographics, referral departments, waiting time, and peak patient hours to support data-driven healthcare decisions.

## Key Insights

- **50.04%** of patients were admitted, while **49.96%** were discharged without admission, indicating a balanced patient admission rate.
- Approximately **59%** of patients were attended within the **30-minute target**, while **41%** exceeded the target waiting time, highlighting opportunities to improve response time.
- The average patient waiting time was **35.3 minutes**, helping monitor Emergency Room operational efficiency.
- Patients reported a high average satisfaction score of **4.99/5**, indicating a positive overall patient experience.
- **General Practice** received the highest number of referrals, making it the busiest referral department.
- Patient arrivals remained consistent throughout the week, helping identify staffing requirements and peak operational hours.
- Dashboard analysis by **age group, gender, race, admission status, referral department, day, and hour** provides a comprehensive view of Emergency Room operations.
